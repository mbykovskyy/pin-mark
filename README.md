# pin-mark
[Polymer][polymer] web component pin mark with an origin at the end of its tip.

## Install

```bash
bower install --save mbykovskyy/pin-mark
```

## Usage

```html
<link rel="import" href="pin-mark.html">

<pin-mark></pin-mark>
```

See [component page][pin-mark] for details and demo.
